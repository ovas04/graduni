# Documentos oficiales de titulación

Consulta estos recursos antes de iniciar tu expediente. Cada facultad puede exigir anexos propios; confirma con tu coordinación académica.

## Normativa base

- [Reglamento de Grados y Títulos de la UNI (2023)](https://www.uni.edu.pe/normas/reglamento-grados-titulos)
- [Directiva de trabajos de investigación UNI](https://www.uni.edu.pe/normas/directivas/trabajos-investigacion)
- [Lineamientos de originalidad y Turnitin](https://www.uni.edu.pe/biblioteca/turnitin)

## Modalidad tesis

- [Formato de registro de tema de tesis](https://www.uni.edu.pe/sites/default/files/documentos/formato-registro-tesis.pdf)

- [Guía de presentación de tesis](https://www.uni.edu.pe/sites/default/files/documentos/guia-presentacion-tesis.pdf)

## Modalidad experiencia profesional

- [Formato de acreditación de experiencia profesional](https://www.uni.edu.pe/sites/default/files/documentos/formato-experiencia-profesional.pdf)
- [Directiva de trabajos de suficiencia profesional](https://www.uni.edu.pe/sites/default/files/documentos/directiva-suficiencia-profesional.pdf)
- [Formato de informe técnico profesional](https://www.uni.edu.pe/sites/default/files/documentos/formato-informe-tecnico.pdf)

:::warning Requisito mínimo
Para la modalidad de experiencia profesional se exige acreditar, como mínimo, tres años de trabajo comprobado después de obtener el grado de bachiller.
:::
